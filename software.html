Conceito de Softwares

1. Sistemas Operacionais

O que são sistemas oc e qual a sua função em um computador?

Um Sistema Operacional (SO) ou Operating System (OS) em inglês é o software principal que "dá vida" ao computador, servindo como o intermediário essencial entre você (usuário)

1.1 Sistemas Livres

O que são sistemas operacionais livres

Sistemas operacionais (SO) livres são softwares que garantem aos usuários a liberdade de executar, estudar, modificar e distribuir o programa. Ao contrário de sistemas proprietários (como o Windows), onde o código é fechado e controlado por uma empresa, um SO livre permite que qualquer pessoa veja como ele funciona e faça melhorias

1.1.1 Linux

O que é o Linux e quais são suas principais características?

O Linux é um sistema operacional de código aberto (open source) baseado no Unix

1.1.2 Distribuições O que são distribuições Linux?

Cite exemplos.

Uma distribuição Linux é um sistema operacional completo que une o motor (kernel) Linux a uma coleção de softwares e interfaces visuais para o usuário final, como o Ubuntu (fácil para iniciantes), o Kali (focado em segurança) e o Mint (similar ao Windows)

1.2 Sistemas Proprietários

O que são sistemas operacionais proprietários?

são softwares com código-fonte fechado e protegido por direitos autorais, pertencentes a uma empresa que restringe sua cópia, modificação ou redistribuição, como o Windows (Microsoft) e o macOS (Apple).

1.2.1 Apple

Quais são os sistemas operacionais desenvolvidos pela Apple e suas características?

Sistemas operacionais proprietários da Apple são softwares de código fechado e uso exclusivo em seus dispositivos, conhecidos pela alta integração e design, como o macOS (computadores), o iOS (iPhones) e o watchOS (Apple Watch).

1.2.2 Microsoft

Quais são os principais sistemas operacionais da Microsoft?

A Microsoft desenvolve a linha Windows, sistemas proprietários líderes em computadores pessoais e servidores, com exemplos como o Windows 11 (uso doméstico e atual), o Windows 10 (versão anterior estável) e o Windows Server (focado em redes empresariais).

2. Softwares

O que são softwares e qual sua importância?

Software é o conjunto de instruções lógicas que controla o hardware, sendo fundamental por dar utilidade às máquinas e possibilitar a automação de tarefas no dia a dia.

2.1 Tipos de Licenças

O que são licenças de software?

As licenças de software são contratos ou termos de uso que definem as regras, permissões e restrições sobre como um programa pode ser utilizado, copiado ou modificado.Sua importância reside em garantir a proteção jurídica do autor e estabelecer os limites legais para o usuário (como o número de dispositivos permitidos ou se o código é aberto para edições).

2.1.1 Licenciados

O que são softwares licenciados?

Softwares licenciados são programas cujo uso é autorizado mediante a aceitação de um contrato de licença, que estabelece termos específicos entre o desenvolvedor e o usuário.

2.1.2 Livres

O que são softwares livres?

Softwares livres são programas que respeitam a liberdade dos usuários, permitindo que qualquer pessoa possa usar, estudar, modificar e distribuir o software sem restrições.

2.1.3 Proprietários

O que são softwares proprietários?

Softwares proprietários são programas protegidos por direitos autorais em que o dono (empresa ou desenvolvedor) mantém controle total sobre o código-fonte, restringindo o que o usuário pode fazer.

2.2 Tipos de Softwares

Quais são os principais tipos de softwares?

Os principais tipos de softwares são os de sistema (gerenciam o hardware), os de aplicação (executam tarefas do usuário) e os de programação (usados para criar novos programas).

2.2.1 de Sistemas

O que são softwares de sistema? Dê exemplos.

Softwares de sistema gerenciam o hardware e sustentam outros programas, incluindo o sistema operacional, o firmware e os drivers, como ocorre no Windows, Android e Linux.

2.2.2 Aplicativos

O que são softwares aplicativos? Cite exemplos.

Softwares aplicativos são programas desenvolvidos para realizar tarefas específicas para o usuário final, como os navegadores (Chrome), editores de texto (Word) e redes sociais (Instagram).

2.2.3 de Programação

O que são softwares de programação? Para que servem?

Softwares de programação são ferramentas utilizadas por desenvolvedores para escrever, testar e criar outros programas e sistemas

2.3 Software para Diagnóstico

O que são softwares de diagnóstico e para que são utilizados?

Softwares de diagnóstico são ferramentas projetadas para monitorar, analisar e testar o funcionamento do hardware e do sistema, sendo utilizados para identificar falhas, medir o desempenho e prevenir erros técnicos.

2.3.1 Diagnóstico de Software

Como é feito o diagnóstico de software?

O diagnóstico de software é realizado por meio de um processo técnico que utiliza ferramentas e métodos específicos para identificar falhas e otimizar o desempenho do sistema.

2.3.2 Diagnóstico de Hardware

Como é feito o diagnóstico de hardware?

através de inspeções visuais, testes de estresse e ferramentas de software especializadas.

2.4 Ferramentas de Busca e Download

O que são ferramentas de busca e download? Cite exemplos.

Ferramentas de busca e download servem para localizar conteúdos na web e transferi-los para o armazenamento local, como ocorre com o Google ferramenta de busca e o uTorrent de download.
